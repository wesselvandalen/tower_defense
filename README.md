# Tower Defense (in Rust)

A simple **Tower Defense** game written in Rust. Place towers, fend off waves of monsters, and defend your base on different kind of maps!

## Installation

Make sure you have [Rust](https://www.rust-lang.org/tools/install) installed. Then, clone the repository and run the project using Cargo:

```bash
git clone https://github.com/wesselvandalen/tower_defense.git
cd tower_defense
cargo run
```

## Dependencies

The project uses various Rust crates. You can install them automatically when building with Cargo.
